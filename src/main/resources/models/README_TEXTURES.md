# Тестовые модели для проверки текстур

В этой папке находятся модели с UV-координатами для тестирования наложения текстур.

## Модели с UV-координатами:

1. **cube.obj** - Куб с UV-координатами (уже был в проекте)
2. **plane.obj** - Плоскость с UV-координатами (уже была в проекте)
3. **square_textured.obj** - Простой квадрат, идеален для первого теста текстур
4. **cube_textured.obj** - Куб с правильной разверткой UV для каждой грани
5. **pyramid_textured.obj** - Пирамида с UV-координатами
6. **cylinder_textured.obj** - Цилиндр с UV-координатами (более сложная модель)

## Как использовать:

1. Загрузите любую из этих моделей через меню "File" -> "Load Model..."
2. Нажмите кнопку "Load Texture..." и выберите изображение (PNG, JPG и т.д.)
3. Включите чекбокс "Use Texture"
4. Модель должна отображаться с наложенной текстурой

## Рекомендации для тестирования:

- **square_textured.obj** - самый простой вариант для первого теста
- **cube.obj** или **cube_textured.obj** - хороши для проверки всех граней
- **plane.obj** - отлично подходит для проверки перспективной коррекции UV

## Формат UV-координат:

Все модели используют стандартный формат OBJ:
- `v` - вершины
- `vt` - UV-координаты (текстурные координаты)
- `vn` - нормали
- `f` - грани в формате `vertex/texture/normal`

UV-координаты обычно в диапазоне [0, 1], где:
- (0, 0) - левый нижний угол текстуры
- (1, 1) - правый верхний угол текстуры
